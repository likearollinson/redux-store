## Motivation

The goal of this project is to showcase using redux to manage state on a webpage. Given base code that uses Context API and manipulate the code so that Redux is used instead.

#### Technologies

- JavaScript
- Node.js
- React
- Redux
- Apollo Server Express
- NoSQL/Mongo
- JSON Web Tokens
- Stripe
